Descripción del Proyecto

Desarrollamos un sistema de gestión integral para una clínica odontológica utilizando Java y NetBeans como entorno de desarrollo, y SQL Server como base de datos. Este proyecto, realizado en un mes, fue llevado a cabo por un equipo de 8 personas, en el cual fui el líder del grupo y el principal enlace con el cliente.

Objetivo:

El objetivo del proyecto era digitalizar y optimizar la gestión de la clínica, permitiendo un manejo más eficiente de los registros de pacientes, citas y documentación clínica.

Tecnologías Utilizadas:

  • Lenguaje de programación: Java
  • Entorno de desarrollo: NetBeans
  • Base de datos: SQL Server
  • Disenos: Photoshop

Funcionalidades Principales:

  • Registro de clientes
  • Historial clínico para niños y adultos
  • Sistema de gestión de citas
  • Impresión de constancias médicas e incapacidades

Rol y Responsabilidades:

  • Coordinar todas las gestiones con el cliente
  • Recopilar y analizar los requerimientos
  • Realizar bocetos y presentarlos al cliente para su aprobación
  • Facilitar la comunicación y coordinación entre el equipo y el cliente

Proceso de Desarrollo:

  1. Recopilación de Requisitos: Realizamos reuniones iniciales con el cliente para entender sus necesidades y recopilar información detallada.
  2. Análisis y Diseño: Analizamos la información y creamos un boceto del sistema, el cual fue aprobado por el cliente.
  3. Desarrollo: Implementamos el sistema según las especificaciones aprobadas, realizando reuniones periódicas para resolver dudas y ajustar el sistema según las nuevas necesidades del cliente.
  4. Pruebas y Ajustes: Probamos el sistema en diferentes entornos para asegurar su correcto funcionamiento.
  5. Entrega Final: Presentamos el sistema terminado al cliente, quien quedó muy satisfecho con el resultado, destacando la eficiencia y claridad en el manejo de los datos comparado con su anterior método manual.

Desafíos y Soluciones:

El principal desafío fue manejar y almacenar correctamente una gran cantidad de datos de manera organizada. A través de reuniones y trabajo colaborativo, logramos encontrar las mejores prácticas para gestionar la información eficientemente.

Resultados y Aprendizajes:

El proyecto fue exitoso, y el cliente quedó encantado con la digitalización de sus procesos. Este proyecto me enseñó la importancia de entender las necesidades del cliente y traducirlas en un sistema funcional y fácil de usar. Además, destacó la importancia del trabajo en equipo y la comunicación efectiva para el éxito de un proyecto.
