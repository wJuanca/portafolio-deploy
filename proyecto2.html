Descripción del Proyecto

Es una tienda online de ropa y accesorios con un enfoque único en el estilo furista. Este proyecto fue desarrollado por un equipo liderado por mí, con el objetivo de ofrecer a los usuarios una experiencia de compra excepcional a través de una aplicación móvil.

Desarrollo del Proyecto:

  1. Ideación y Planificación: Iniciamos con una lluvia de ideas para determinar la mejor opción para nuestra tienda online.
  2. Diseño y Funcionalidad: Nos reunimos para crear un boceto detallado de la aplicación, definiendo todas las funcionalidades y secciones que debería incluir. Con esta hoja de ruta clara, comenzamos con la programación.
  3. Desarrollo Iterativo: Dividimos el trabajo en pequeños grupos y nos reunimos regularmente para mantenernos al tanto del progreso. Como líder del equipo, supervisé el desarrollo de cada sección, brindando apoyo y recursos según fuera necesario.
  4. Implementación de Base de Datos: Utilizamos una base de datos online, Firebase, para almacenar los datos de los usuarios, como nombres de usuario y contraseñas. Esto garantizó un acceso seguro y confiable a la aplicación.

Resultado y Aprendizajes:

El desarrollo de Furista fue una experiencia enriquecedora que me permitió explorar el vasto mundo del desarrollo móvil. Aprendí que hay una amplia gama de tecnologías y herramientas disponibles, y Flutter demostró ser una excelente opción para crear aplicaciones multiplataforma de manera eficiente y atractiva.

Duración del Proyecto:

Tardamos aproximadamente tres semanas en completar el desarrollo de la aplicación. Aunque fue un desafío, el proceso fue dinámico y gratificante, y el resultado final fue una aplicación atractiva y funcional que también se puede acceder desde una página web con los mismos elementos.
